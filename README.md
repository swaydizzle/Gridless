# Gridless
Gridless is an off grid LoRa protocol communication device.

This project utilizes the ESP32-S3 module, SX1276 LoRa module and 3.5 in. TFT touch display to create a handheld battery powered off grid communication device.

C++ is used in conjunction LoRa, TFT_eSPI, LVGL and BLE libraries to develop firmware for messaging and BLE connection.
